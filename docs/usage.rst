=====
Usage
=====

To use syncalotypus in a project::

    import syncalotypus


Config options 
================ 

The following can be set in your env vars or in a `.env` file.

+--------------------------+------------+-----------+
| Variable                 | Default    | Function  |
+==========================+============+===========+
| SNYCALOTYPUS_CONFIG_FILE | `~/.syncalotypus.yml` | Configure sync locations. |
+------------+------------+-----------+
| body row 2 | Cells may span columns.|
+------------+------------+-----------+
| body row 3 | Cells may  | - Cells   |
+------------+ span rows. | - contain |
| body row 4 |            | - blocks. |
+------------+------------+-----------+
